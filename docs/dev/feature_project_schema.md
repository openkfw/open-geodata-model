**Title:** GeoJSON Feature

|                           |                  |
| ------------------------- | ---------------- |
| **Type**                  | `object`         |
| **Required**              | No               |
| **Additional properties** | Any type allowed |

**Description:** Main Schema with geometries and [project_data/project_core_schema](project_core_schema.md) properties. It is built hierarchially and is represented as Feature - a known Geojson Format. <br> In the properties are all the project information which is separated from the geometry.

| - [id](#id )                 | No      | Combination      | No         | -                           | -                      |
| + [properties](#properties ) | No      | object           | No         | In project_core_schema.json | Project Location Model |
| + [geometry](#geometry )     | No      | Combination      | No         | -                           | -                      |
| - [bbox](#bbox )             | No      | array of number  | No         | -                           | -                      |
